    날은 따뜻해지고 있지만

    2018/04/10

    나의 마음도 따뜻해지고 있는지 모르겠다. 오히려 더 냉정해지고 있지만 사춘기의 아이처럼 짜증이 많이 밀려온다. 오늘 링크드인을 통해서 2곳 지원했다. 지원한 분야는 데이터 엔지니어 직군이다. 데이터 엔지니어 직군은 다음의 기술요소를 요구하는 것 같다.

    분산 시스템 이라고 되어 있다. Production 수준에서 데이터분야의 기술은 다음일 것이다.

    • Apache Spark
    • Apache Kafka
    • Apache Hadoop

    그리고 해당 플랫폼 상에서 구현되는 언어는 Scala, Java, Python 순이다. 이 기술들이 무엇인지 알지만 Hans-on을 안 한지 오래되서 사실 감이 많이 없다. 그래서 결국 현재 Apache Spark 공부 중인데, 나이 탓 하고 싶지 않지만 확실히 감이 많이 떨어진 것 같다. 그래서 한편으로 무섭고 두렵다.

    또한 On-Site Interview에서 코딩테스트도 볼 것 같은데, 걱정도 되고 두렵기도 하고 긴장이 많이 된다. 코딩테스트도 마찬가지로 준비해야 해서 사실 쉽지 않다. 회사를 그만두기로 했다. 하지만 어떻게 그만 둘 지는 고민해야 하는 것 같다. 다음의 회사를 결정하지 않고 회사를 그만둔 적이 없어서 많이 두렵다.

    하지만, 결정할 수 있는 시간이 이제는 많이 없다. 미루기도 애매한 나이다. 지금 돌아가는 회사 구조상 더 이상 내 할일은 없는 것 같다.

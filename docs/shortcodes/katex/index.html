    KaTeX

    KaTeX shortcode let you render math typesetting in markdown document. See KaTeX

    Example

    {{< katex [display] [class="text-center"]  >}}
x = \begin{cases}
   a &\text{if } b \\
   c &\text{if } d
\end{cases}
{{< /katex >}}
    \( x = \begin{cases} a &\text{if } b \\ c &\text{if } d \end{cases} \)

    Display Mode Example

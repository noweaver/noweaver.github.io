    맥 키보드 한/영 키 설정

    2020/01/25
    기록
    Mac, Karabiner

    Karabiner 설치 여기서는 Brew로 맥의 GUI 프로그램을 설치하는 brew cask방법으로 설치한다. $ brew cask install karabiner-elements Karabiner 구성파일 셋업 출처: 작은곰의 작업실 ⌘ command + ⇧ shift + .을 누르면 숨겨진 파일을 보여준다. ~/.config/karabiner/assets/complex_modifications 의 경로를 찾아 이동한다. 혹은 ⌘ command + ⇧ Shift + g 실행해서 위의 경로를 입력해서 이동한다. 애플키보드 한영전환/한자키 지정 (rev.1.13d) 다른 이름(KorEng-Hanja.json)으로 저장해서 위의 디렉터리로 파일 복사 혹은 이동한다.
